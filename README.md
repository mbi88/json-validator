
Based on <a href="https://github.com/everit-org/json-schema">JSON Schema Validator</a>.

Validates json schema with something of:
- `org.json.JSONObject`
- `org.json.JSONArray`
- `io.restassured.response.Response` 
- `java.lang.String`
